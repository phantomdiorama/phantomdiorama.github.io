BLOGS

Uses

Here is a list of the hardware and software I use. Might be interesting to someone.

Desktop (2020): Ryzen 3200G, 8GB RAM, SSD. I had my previous desktop for a little more than a decade so this is a pretty big upgrade in terms of speed and hardware stability. This was a present to myself after finishing uni.

HP 250 G6 (2018): Intel i5, 4GB RAM, SSD. Current laptop. Only gets occasional use lately.

Raspberry Pi Keyboard: See above. Cheap and works as a USB hub.

Roccat Kone AIMO: Nice mouse. lots of clicky buttons and RGB if you like that sort of thing.

iPod Touch (7th Gen): An iPhone without the phone, in other words: perfect. Use this for twitter and writing in bed.

Software

Windows 10: Previous desktop OS was Win 7. This is like a slightly worse version of that. Works fine.

Ubuntu 20.04: The Win 10 install on the laptop died from neglect, so now there’s this. It’s reasonably fast and works similarly enough to Windows. Has a bug where it locks up on shutdown which I should fix at some point.

Notepad++: Where the majority of writing happens. Rock solid.

Word/Scrivener/1Writer: Various writing tools. Scrivener is nice, but I haven’t dived into it too deeply. Word is sadly a necessity. On iOS, 1Writer is the best text editor and I’ve tried them all.

Dropbox: For backup and sync. Only way I’ve found to reliably sync between iOS and Windows. Also use File History and rclone for various backup purposes.

zola: For building this website. Nice and simple.
