Uses

November 26, 2022

Lately, when I’m not procrasti-scrolling Twitter or trying not to drown, I’ve been watching a lot of desk setup videos on youtube and reading back through Uses This. Here is my attempt at that—a list of the hardware and software I use. Might be interesting to someone.

hardware

Recently, if you can called pre-pandemic recent, I finished my studies at university and as a present to myself I replaced the 11(!) year old desktop I had been using. I didn’t go over the top on specs: the new one is a low-end Ryzen. But it’s an enormous set up in terms of speed and stability. The biggest upgrade was going from a spinning platter hard drive to an SSD. It makes everything snappy. I chose to go without a dedicated video card this time around because I wanted this PC to be more of a work device, and I can’t be distracted by games if my machine cannot run them. For input, the best keyboard for my hands is, surprisingly, from Raspberry Pi. It’s light, works as a USB hub and you can colour it in. The arrow keys are terrible though. I have a second one for co-writing. My mouse is a Logitech G305. Again it was cheap, but good. And it’s white to match the keeb. For the display I’m using an old Sony Bravia 32” TV. This was left over from the old PC and works pretty well. There’s also a Nintendo Switch plugged into it. For Animal Crossing.

My most productive device is an iPod Touch which is basically an iPhone without the whole receiving calls nonsense, in other words, it’s perfect. Because of my bad back, a lot of writing happens while I’m horizontal. And the iPod is the best device I’ve found for that. It’s small, light and performant enough. I’ve had a few of them over the years. Of course, Apple has discontinued it. And the next cheapest fruit device costs roughly double what it cost. So I’ll have to look beyond the orchard when it comes time to replace it.

Non-work tech includes a Raspberry Pi 3B that I use to push internet radio or Spotify to some outdoor speakers. And a Kindle Paperwhite for reading.

software

My software needs are fairly minimal. I use Windows 10 because that’s what the new computer came with. Previously I used Windows 7. Which I liked. This is like a slightly worse version of that. It works fine.

Writing happens in Vim or 1Writer. Vim took a heckin’ lot of time to configure, but I’ve managed to smoosh it into a shape that makes my terrible hands happy. So it’s worth it. Sadly, writing collaboratively requires either Google Docs or Word, so I’m stuck with them. Dropbox keeps everything backed up and is the only way I’ve found to reliably sync between iOS and Windows.

Beyond work, there’s a bunch of stuff I used regularly:

  • Firefox because there’s nothing better.
  • 7-Zip File Manager - dual-pane file management.
  • Spotify and gPodder for music and podcasts.
  • VLC for watching stuff.
  • File History - Like Time Machine but for Windows and built-in. Rclone is also nice for cross-platform backup.
  • Hugo - for building this website. I used Zola previously but it fell over somehow. Hugo works well enough.
  • Autohotkey - create custom keyboard shortcuts. Can even do auto-correction. Essential.
  • Github Desktop - very nice git interface.
  • Todoist - A web-based to do list.
  • Tweetdeck - Better twitter interface. For now. The new version currently in preview looks horrible.
  • And Facebook for voice chatting while playing Animal Crossing.

dream setup

I’ve pretty much got it. I think. For now. I would like to replace the Sony with an actual monitor with higher resolution and a less ridiculous bezel. My eyes would probably like that.
