things I use

I enjoy reading Uses This. So here is my version of that—a list of the hardware and software I use. The things I need list would likely be shorter. Anyway, this might be interesting to someone.

Last updated 2023-09-10.

computer

Recently, I replaced my 11 year old desktop with a modest Ryzen machine that I hope to run for at least as long. And really, there’s no convincing reason a computer should not remain functional and useful for a decade. But I won’t get into that here. For input, the best keyboard I’ve found for my hands is, surprisingly, from Raspberry Pi. My mouse is a Logitech. And so is my Webcam.

other hardware

A lot of writing happens while I’m horizontal. And the best device I have found for that is an iPod Touch. It’s an iPhone without the phone calls. It’s perfect. And discontinued. I also have a Kindle for reading and a Nintendo Switch for Animal Crossing.

software
